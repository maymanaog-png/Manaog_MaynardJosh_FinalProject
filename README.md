Expense Splitter System
By: Maynard Josh R. Manaog

Description
The Expense Splitter System is a Python-based command-line application designed to help groups manage and divide shared expenses fairly and efficiently. It allows users to add people, record expenses, view balances, and generate settlement plans automatically.

The program demonstrates important programming concepts such as:
- Object-Oriented Programming (OOP)
- File Handling using JSON
- Functions and Classes
- Loops and Conditional Statements
- Error Handling and Data Management

This system is useful for trips, shared apartments, outings, and other group activities involving shared expenses.

Features

- Add new group members
- Record shared expenses
- View balances of all members
- Generate settlement plans
- Save expense data using JSON
- Exit the program safely

1. Install Python
Download and install Python 3.10
2. Clone the Repository
git clone https://github.com/maymanaog-png/Manaog_MaynardJosh_FinalProject.git cd

Sample CLI Usage

Adding a Person
EXPENSE SPLITTER

1. Add Person
2. Add Expense
3. Balances
4. Settle Debts
5. Save/Exit
0. Exit

Choice: 1
Name: Alice
Added Alice

---

Adding an Expense
Choice: 2

Expense:
Description: Dinner
Amount: $60
Paid by: Alice
Split (comma/Enter=all): Bob, Charlie

Dinner: $60.00 (paid by Alice)

---

Viewing Balances
 
 BALANCES:
---------------------------------------------
Alice      $+40.00 (is owed $40.00)
Bob        $-20.00 (owes $20.00)
Charlie    $-20.00 (owes $20.00)

---
